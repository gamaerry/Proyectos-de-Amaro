1) 
la linea 
'var estaExpandido by remember { mutableStateOf(false) }'
 es una abreviacion para
'var estaExpandido: MutableState<Boolean> = remember { mutableStateOf(false) }' 
en donde la variable boolean está envuelto en un 'MutableState' que la implementa de manera especial (a la variable), por ello podemos expresar explicitamente su tipo como 
'var estaExpandido: Boolean by remember { mutableStateOf(false) }'
(al declararse como un MutableState necesitaríamos acceder a su valor explicitamente con 'estaExpandido.value', ie la palabra clave 'by' simplifica la notación '.value')

2)
el modificador Modifier.fillMaxSize() se coloca en el setContent del onCreate y no en ContenidoPrincipal() (cualquier otro componible secundario) para que no se renderice la pantalla completa en el preview pero sí en el onCreate() de la app

5)

11) en el fragmento:
LazyColumn {
    items(items = lista, key = { objeto -> objeto.id }) { objeto -> 
	ItemObjeto(objeto)
    }
}
el key se especifica por cuestiones de optimización pues internamente sin un key o id cada item se le identifica por su posición en la columna, es decir que si hay una modificacion que altere esa posición en varios items se aplicaran multiples cambios innecesarios que no serán buenos para el rendimiento
