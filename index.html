Aviso de Privacidad

Rompecocos
Última actualización: 23 de noviembre de 2025

Responsable: gamaerry (Desarrollador independiente, México)

1. Información que se recopila

Rompecocos no recopila, almacena ni transmite ningún tipo de datos personales. La aplicación no solicita permisos especiales, no accede a información del dispositivo, no usa cookies, no utiliza servicios de analítica ni recopila datos sensibles o de ningún tipo.

2. Uso de la información

Al no recopilar información, no existe ningún uso, tratamiento o transferencia de datos personales.

3. Servicios de terceros

La aplicación no integra APIs, SDKs ni servicios de terceros que recopilen información.

4. Seguridad

Dado que no se recopila información, no se requiere ningún mecanismo de resguardo o protección de datos personales.

5. Cambios al Aviso de Privacidad

Este Aviso de Privacidad puede sufrir modificaciones, cambios o actualizaciones en cualquier momento. Los cambios se publicarán en:

https://gamaerry.github.io/Proyectos-de-Amaro/
